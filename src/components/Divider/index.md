# Divider 分割线

<p>分隔内容的文档分割线</p>

## 何时使用

- 对不同章节的文本段落进行分割。

- 对行内文字/链接进行分割，例如表格的操作列。

## 水平分割线

默认为水平分割线，可在中间加入文字。

## 带文字的分割线

分割线中带有文字，可以用 orientation / left / right 指定文字位置。使用 left / right 时， orientation 无效。
